AetherMind API

Welcome to the AetherMind API documentation. AetherMind is a Developmental Continual Learning Architecture (DCLA) that provides state-of-the-art reasoning and learning capabilities. Our API allows you to integrate this power into your own applications.

Authentication

All API requests must be authenticated with an API key. You can generate an API key from your user dashboard after logging in with GitHub.

Include your API key in the Authorization header of your requests:

Authorization: Bearer YOUR_API_KEY

Quickstart: First Chat

Here's a quick example using curl to send your first chat message to AetherMind:

curl -X POST https://api.aethermind.ai/v1/chat/completions \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "model": "aether-pro",
  "messages": [
    {
      "role": "user",
      "content": "Explain active inference in simple terms."
    }
  ]
}'

Chat Completions

The main endpoint for interacting with AetherMind. It's OpenAI-compatible, making it easy to switch from existing integrations.

POST /v1/chat/completions

Body Parameters:

  • model (string, required): The model to use. Currently aether-pro.
  • messages (array, required): A list of message objects, each with role ('user', 'assistant') and content.
  • stream (boolean, optional): Whether to stream the response back.

Episodic Memory Search

Search through the agent's episodic memory to find relevant past interactions or learned concepts.

POST /v1/memory/search

Body Parameters:

  • query (string, required): The natural language query to search for.
  • top_k (integer, optional): The number of results to return. Defaults to 5.

ToolForge: Runtime Tool Creation

Dynamically create and register new tools for the agent to use during its reasoning cycles. This allows you to extend AetherMind's capabilities on the fly.

POST /v1/tools/create

Body Parameters:

  • name (string, required): The name of the tool (e.g., 'get_weather').
  • description (string, required): A description of what the tool does.
  • parameters (object, required): A JSON schema object describing the tool's parameters.
  • code (string, required): The Python code to execute for the tool.

Python SDK

Our Python SDK provides a convenient wrapper around the API.

pip install aethermind

Example usage:

from aethermind import AetherMindClient

client = AetherMindClient(api_key="YOUR_API_KEY")

response = client.chat.completions.create(
    model="aether-pro",
    messages=[{"role": "user", "content": "Hello, world!"}]
)

print(response.choices[0].message.content)

JavaScript SDK

Our JavaScript/TypeScript SDK for web and Node.js applications.

npm install @aethermind/sdk

Example usage:

import { AetherMindClient } from '@aethermind/sdk';

const client = new AetherMindClient({ apiKey: 'YOUR_API_KEY' });

async function main() {
    const response = await client.chat.completions.create({
        model: 'aether-pro',
        messages: [{ role: 'user', content: 'Hello, world!' }],
    });
    console.log(response.choices[0].message.content);
}

main();
