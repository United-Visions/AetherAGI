The Architecture

AetherMind separates reasoning from knowledge, enabling infinite learning without retraining.

The Brain (Cognitive Core)

Fixed reasoning engine with core logic priors. Knows how to think, not what to think. Seeded with axioms from seed_axioms.py.

The Mind (Knowledge Repository)

Infinite vector memory (Pinecone). Stores episodic memories, domain knowledge, and experiences. Expands forever through continuous learning.

The Orchestrator (Active Inference)

Minimizes "surprise" through continuous learning. Routes information between Brain and Mind. Manages meta-cognition and self-modification.

The Body (Adapters)

Chat, GitHub, Vision, Smart Home. Interface with any system. Generate new tools autonomously via ToolForge.
