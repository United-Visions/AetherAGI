## üéØ Complete City Build Example

**Goal:** Build a realistic cyberpunk city with 50 buildings, 20 vehicles, and 30 animated NPCs.

### Step 1: Generate Buildings (Meshy.ai)

```xml
<aether-meshy action="generate" prompt="futuristic cyberpunk skyscraper with neon lights">
{
  "style": "realistic",
  "format": "fbx",
  "project_id": "1449261"
}
</aether-meshy>

<aether-meshy action="generate" prompt="cyberpunk office building glass facade">
{
  "style": "realistic",
  "format": "fbx"
}
</aether-meshy>

<aether-meshy action="generate" prompt="neon lit cyberpunk apartment complex">
{
  "style": "realistic",
  "format": "fbx"
}
</aether-meshy>
```

**What happens:**
- Meshy generates 3D models (takes ~2-5 min each)
- Models saved to `/tmp/aethermind_meshy/`
- Returns FBX files with textures

### Step 2: Download Vehicles (SketchFab)

```xml
<aether-sketchfab action="search">
{
  "query": "cyberpunk hover car",
  "downloadable": true,
  "license": "cc",
  "max_results": 10
}
</aether-sketchfab>

<!-- Brain sees results, picks best one -->
<aether-sketchfab action="download">
{
  "model_id": "abc123xyz",
  "project_id": "1449261"
}
</aether-sketchfab>
```

**What happens:**
- Searches SketchFab library
- Downloads GLTF models (ZIP with textures)
- Saved to `/tmp/aethermind_sketchfab/`

### Step 3: Download Characters (Mixamo)

```xml
<aether-mixamo action="batch_download">
{
  "characters": [
    {"name": "Business Man", "animations": ["Walking", "Idle", "Phone Call"]},
    {"name": "Teen Girl", "animations": ["Walking", "Running", "Waving"]},
    {"name": "Police Officer", "animations": ["Walking", "Standing Guard"]},
    {"name": "Street Performer", "animations": ["Hip Hop Dancing", "Breakdance"]},
    {"name": "Cyberpunk Agent", "animations": ["Walking", "Running", "Combat Idle"]}
  ],
  "project_id": "1449261"
}
</aether-mixamo>
```

**What happens:**
- Downloads 5 characters with 15 animations total
- Rigged FBX files with animations
- Saved to `/tmp/aethermind_mixamo/`

### Step 4: Setup PlayCanvas Scene

```xml
<aether-playcanvas action="setup_scene" project_id="1449261">
{
  "scene_type": "cyberpunk_city",
  "lighting": "night",
  "skybox": "urban_night"
}
</aether-playcanvas>
```

**What happens:**
- Creates ground plane
- Sets up night lighting (dim ambient, neon accents)
- Adds main camera at elevated position
- Creates fog for atmosphere

### Step 5: Place Buildings

```xml
<aether-playcanvas action="create_entity" project_id="1449261">
{
  "entity_name": "Building_01",
  "components": ["model", "collision"],
  "position": {"x": 0, "y": 0, "z": 0},
  "scale": {"x": 1, "y": 2, "z": 1}
}
</aether-playcanvas>

<!-- Repeat for 50 buildings in grid layout -->
```

**Note:** Currently creates placeholder entities. Full auto-upload coming next!

### Step 6: Generate NPC Navigation Script

```xml
<aether-forge tool="npc_navigation" action="generate">
{
  "description": "NPC walks along street path with collision avoidance",
  "language": "javascript",
  "params": {
    "speed": 1.5,
    "patrol_points": [[0,0,0], [10,0,0], [10,0,10], [0,0,10]]
  }
}
</aether-forge>

<aether-playcanvas action="upload_script" project_id="1449261">
{
  "script_name": "npcNavigation",
  "script_content": "<!-- Generated by ToolForge -->"
}
</aether-playcanvas>
```

### Step 7: Deploy AetherBridge

```xml
<aether-playcanvas action="deploy_bridge" project_id="1449261">
{
  "api_url": "http://localhost:8000/v1/game/unity/state"
}
</aether-playcanvas>
```

**Result:**
- Game now syncs with AetherMind backend every 500ms
- Brain can control NPCs in real-time
- User can chat: "Make NPCs dance" ‚Üí All NPCs switch to dance animation

## üêõ Troubleshooting

### "Meshy API error: 401"
```bash
# Check API key is valid
curl -H "Authorization: Bearer $MESHY_API_KEY" https://api.meshy.ai/v2/text-to-3d
```

### "SketchFab: No downloadable models found"
- Add `downloadable: true` filter
- Use `license: "cc"` for Creative Commons only
- Search broader terms (e.g., "car" not "red sports car")

### "Mixamo: Login failed"
- Verify Adobe account credentials
- Try non-headless mode: `MIXAMO_HEADLESS=false`
- Check 2FA is disabled on account

### "PlayCanvas: Entity not found"
- Project ID must be correct
- Account must have access to project
- Editor must be open in browser automation
